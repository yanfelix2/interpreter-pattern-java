# Interpreter Pattern - Java

This project demonstrates the implementation of the **Interpreter Design Pattern** in Java.

## 📚 About
The Interpreter Pattern defines a representation for the grammar of a language and provides an interpreter to evaluate expressions in that language.

## 💻 Technologies
- Java

## 📌 Example Expression

(5 + 2) * 3
